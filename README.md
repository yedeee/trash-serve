# trash-serve
A mini python server you can host ig built by me with hate.

## How to set up
1. Clone the repo with git. Install git if you haven't already, then use ` git clone https://github.com/yedeee/trash-serve.git `, or manually download the source code by going to `Code`, then `Download as ZIP`
2. Go to the repo folder
3. Make sure you have python dependencies ffmpeg and watchdog installed. Use pip to install them.
4. Run runserve.py and enter the desired port.
5. If you want, add videos to the video folder.
